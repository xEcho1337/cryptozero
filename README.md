# CryptoZero

Lightweight & modular helper Python library for Cryptography CTF challenges.

## Features

- **Ciphers**: RSA
- **Encodings**: Base64, Hex, URL
- **Attacks**: Fermat, Wiener, Small exponent, Pohlig Hellman
- **Utilities**: Byte manipulation, math helpers (egcd, modinv)

## Installation

```bash
pip install . -e
```
